# UI Primitives Package (`packages/ui`)

## Purpose
A collection of reusable visual components and design tokens for the Vizorix web client.

## Responsibility
- Centralizes core CSS design tokens (typography, colors, gradients).
- Offers reusable visual block primitives (buttons, modals, layout cards).
- Presents custom elements (memory stack containers, heap pointer links, timeline scrollers).

## Planned Public Interfaces
- CSS styles export (`/dist/index.css`)
- Component definitions (e.g., `CodeEditorView`, `MemoryVisualizerView`, `TimelineScrubberView`)

## Dependencies
- None
