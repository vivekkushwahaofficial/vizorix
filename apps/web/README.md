# Vizorix Web Application (`apps/web`)

## Purpose
This application serves as the user-facing web interface for the Vizorix platform.

## Future Responsibility
- **Code Editor**: A web-based editor for students to write, edit, and submit Java programs.
- **Timeline Controls**: Interface elements to play, pause, rewind, and step through execution phases.
- **Memory Visualizations**: Graphic layouts displaying call stack frames, local variables, and heap objects/pointers.
- **AI Explanations**: A panel presenting contextual explanations generated for the currently selected execution step.
